PRACTICAL WEB DEVELOPMENT

Since the commercialization of of the web, web development has been a growing
industry.The growth of this industry is being driven by businesses wishing to use
their websites to sell products and services to customers.

There is open source software for web development like the BerkeleyDB, GlassFish,
LAMP {Linux, Apache, MySQL PHP} stack and Perl.Plack. This has kept the cost of
learning web development to a minimum Another contributing factor to the rise of the
industry has been the rise of easy-to-use WYSIWYG web development software such as
Adobe Dreamweaver, BlueGriffon and Microsoft Visual Studio. Knowledge of HyperText
Markup Language or of programming languages is still required to use such software but
the basics can be learnt and implemented quickly with the help of files, technical books,
internet tutorials or face-to-face training.

An ever growing set of tools and technologies have helped developers build more dynamic
andinteractive websites Further, web developers ow help to deliver applications as web
services which were traditionally only available as applications on a desktop based computer.
This has provided many opportunities to decentralise information and media distribution.
Examples can be seen with the rise of cloud services such as Adode Creative Cloud,
Dropbox and Google Docs. These web services allow users to interact with applications
from many locations instead of being tied to a single work station for their application
environment.
